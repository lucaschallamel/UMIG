UMIG API V2 (2.0.0)

Download OpenAPI specification:

REST API for the Unified Migration and Implementation Governance (UMIG) application.

This API follows RESTful principles and is implemented using ScriptRunner Groovy scripts. It supersedes the V1 API. Architectural principles are documented in ADR-017.
